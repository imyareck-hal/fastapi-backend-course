	1) Ограниченность объемом ОЗУ (условно озу 32гб, а hdd/ssd 1тб)

	3) Стоимость 1гб озу для сервера на хостинге тупо дороже чем 1гб статичной памяти в hdd/ssd
	4) В кейсах с разбивкой хранящихся данных на какие-то подвиды в случае с ОЗУ будет сложнее реализовать логику
